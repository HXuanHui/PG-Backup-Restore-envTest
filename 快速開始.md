# 快速開始指南

## 您的連線資訊

根據您提供的資訊：
- **主機**: 10.31.155.37
- **使用者**: cghadmin
- **密碼**: cgH@Dmin2025

## 使用方法

### 方法一：直接執行（使用預設值）

腳本已經預設了您的連線資訊，可以直接執行：

```bash
python pg_backup_restore_test.py --auto-start
```

這會使用預設的連線資訊並自動開始測試，不需要任何輸入。

### 方法二：使用命令列參數

```bash
python pg_backup_restore_test.py --host 10.31.155.37 --user cghadmin --password cgH@Dmin2025 --auto-start
```

### 方法三：互動式輸入

如果您想手動輸入連線資訊：

```bash
python pg_backup_restore_test.py --interactive
```

### 方法四：使用 SSH 金鑰（如果有的話）

```bash
python pg_backup_restore_test.py --key-file ~/.ssh/id_rsa --auto-start
```

## 命令列參數說明

- `--host, -H`: SSH 主機名稱或 IP（預設: 10.31.155.37）
- `--user, -u`: SSH 使用者名稱（預設: cghadmin）
- `--password, -p`: SSH 密碼（預設: cgH@Dmin2025）
- `--key-file, -k`: SSH 金鑰檔案路徑（如果使用金鑰認證）
- `--port`: SSH 連線埠號（預設: 22）
- `--interactive, -i`: 使用互動式輸入（忽略命令列參數）
- `--auto-start, -a`: 自動開始測試，不需要確認

## 注意事項

1. 確保已安裝依賴：`pip install -r requirements.txt`
2. 測試過程可能需要數小時，請確保有足夠時間
3. 確保網路連線穩定
4. 測試結果會自動儲存到 `test_results_*.json` 和 `test_results_*_report.txt`
